# 今日人品插件 v0.0.4

一个功能丰富的今日人品查询插件，支持群排行榜和AI增强功能。

## 功能特点

- ✨ 每日随机生成人品值（可自定义范围）
- 🏆 群聊人品排行榜
- 🤖 可选的AI模拟占卜过程
- 💬 可选的AI生成建议/吐槽
- ⚙️ 丰富的配置选项

## 使用方法

### 基础指令
- `/jrrp` 或 `-jrrp` 或 `/今日人品` - 查询今日人品
- `/jrrprank` 或 `/人品排行` - 查看群聊人品排行榜（仅群聊可用）

### 配置说明

在插件配置页面可以设置：
- **插件总开关** - 启用/禁用整个插件
- **人品值范围** - 自定义最小值和最大值
- **AI增强功能** - 开启后可获得更有趣的体验
- **自定义提示词** - 调整AI生成内容的风格


## 示例

普通模式：
用户: /jrrp
机器人: 🔮 阿凌 的今日人品值: 88
📊 运势: 好运连连 🎉


AI增强模式：
用户: /jrrp
机器人: 水晶球泛起神秘波纹，星光汇聚...阿凌今日人品88分，好运连连！
机器人: 建议：今天适合抽卡和买彩票，欧皇附体！
