# AstrBot 今日人品插件

一个有趣的人品值查询插件，让你的Bot更加生动活泼！

## 功能特性

- 🔮 **每日人品查询** - 每个用户每天可以查询一次人品值
- 🎯 **智能权重系统** - 可自定义人品值范围和出现概率
- 🤖 **LLM增强体验** - 支持使用AI生成有趣的查询过程和建议
- 📊 **排行榜功能** - 查看今日人品排行榜
- 📜 **历史记录** - 查看个人的人品历史

## 使用方法

### 基础指令
- `/jrrp` 或 `-jrrp` 或 `/今日人品` - 查询今日人品
- `/jrrp_rank` - 查看今日人品排行榜
- `/jrrp_history` - 查看个人人品历史

### 示例

用户: /jrrp
机器人: 🔮 水晶球中迷雾缭绕...你的今日人品值是：88！
机器人: 💭 88分的欧皇，今天适合抽卡！


## 配置说明

插件支持丰富的配置选项：

- **LLM功能开关** - 控制是否使用AI生成内容
- **自定义提示词** - 可以修改AI生成内容的风格
- **人品值范围** - 自定义不同分数段的描述和权重

详细配置请在AstrBot管理面板的插件配置页面进行设置。
